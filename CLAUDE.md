# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Git & GitHub Workflow

- Primary branch: `main`
- Remote: https://github.com/yardendance-bot/ClaudeCodeTest
- **Commit and push frequently.** After every meaningful unit of work — a new file, a bug fix, a feature addition — stage, commit, and push to `origin main`. Do not batch multiple unrelated changes into one commit.
- Write clean, descriptive commit messages: a short imperative subject line, with a body if context is needed.
- Use `git push origin main` after every commit so the remote always reflects the current state of work.
